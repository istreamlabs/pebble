
## Brand

Brand colors are to be used sparingly.

```js noeditor
<ColorSwatch name="$brand-orange" value="#F04E23" />
<ColorSwatch name="$black" value="#000000" />
<ColorSwatch name="$white" value="#FFFFFF" />
```

## UI Neutrals

Neutral colors are used for text, backgrounds, lines and borders.

```js noeditor
<ColorSwatch name="$neutral-100" value="#FAFBFB" />

<ColorSwatch name="$neutral-300" value="#DFE3E8" />

```
